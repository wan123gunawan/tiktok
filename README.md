🔥 TikTok Product Research App

Aplikasi berbasis Streamlit untuk riset produk viral menggunakan data dari Google Trends.

App ini membantu kamu menemukan produk yang sedang naik tren dan berpotensi viral untuk dijual di TikTok / marketplace.

🚀 Fitur Utama
✅ Input banyak keyword sekaligus
📈 Grafik tren pencarian
🔍 Data langsung dari Google Trends
🔑 Related queries (keyword viral)
📊 Analisa otomatis:
🔥 NAIK
⚠️ TURUN
😐 STABIL
💯 Viral Score (0–100)
⏱ Filter waktu:
Preset (7 hari, 1 bulan, dll)
Custom tanggal
